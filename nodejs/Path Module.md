
📂 **Stop hardcoding file paths in Node.js. Use the `path` module instead.**

The `path` module helps you work with file and directory paths safely across Windows, macOS, and Linux.

Some methods you'll use all the time:

✅ `path.join()` → Join path segments correctly
✅ `path.resolve()` → Create an absolute path
✅ `path.basename()` → Get the file name
✅ `path.dirname()` → Get the parent directory
✅ `path.extname()` → Extract the file extension
✅ `path.parse()` → Break a path into useful parts

💡 Pro tip: Avoid manually concatenating paths with `/` or `\\`. Let the `path` module handle platform differences for you.

Clean, portable code starts with the right utilities. 🚀
